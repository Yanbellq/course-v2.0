Для того щоб запустити проект потрібно:
1. мати локальний сервер mongo, та щоб він був запущений
2. потрібно мати python3.13+
3. потрібно мати nodeJS latest LTS version
4. потрібно створити віртуальне середовище, в кореневій папці проекта виконати команду в терміналі --> python -m venv .venv 
4. потрібно активувати середовище --> 
	-- Windows: source .venv/Scripts/activate
	-- Linux / MacOS: source .venv/bin/activate
5. потрібно запустити файл build.sh  --> ./build.sh
6. можна запустити проект --> python manage.py runserver


PS. Якщо у вас Windows, то всі дії виконуйте у терміналі gitbash



Для успішного користування потрібно:
1. використовуючи MongoDB Compass підключитись до локального серверу та моніторити базу данних
2. після створення першого користувача змінити йому роль на адміна(role: "admin"), бо програмою задумувалось що адмін всьоголиш один який робиться на самому початку і його реєстування без прямого створення в бд неможливе. 
3. за бажанням можна вручну створити колекції або після перших створених об'єктів вони автоматично створяться, колекції продуктів та категорій можна імпортувати з бекапа
